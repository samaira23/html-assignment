Introduction to HTML

HTML stands for HyperText Markup Language. It is used to create web pages and web applications.

Special characters can be included in HTML using HTML entities. Here are some examples:

  • & - represents the ampersand symbol (&)
  • < - represents the less-than symbol (<)
  • > - represents the greater-than symbol (>)
  • © - represents the copyright symbol (©)
  • ® - represents the registered trademark symbol (®)
  • ™ - represents the trademark symbol (™)
